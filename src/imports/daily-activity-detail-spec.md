[목표]
일상 트랙에서 “추천만 하고 끝”이 아니라,
사용자가 오늘 1개 행동을 하고 → 완료 체크 → 짧은 피드백 기록 → 다음 날 비교
루프가 자연스럽게 닫히도록 디테일 화면을 추가한다.
과잉 입력은 금지(노년층 친화).

[추가/강화 화면 — 08_Detail_Spec 페이지에 생성]
1) P-Home-Daily-02_TodayPlan
- 제목: “오늘의 계획”
- CMP/Checklist/DailyPlan (최대 3개)
- 기본은 1개만 추천되어 들어가고, 사용자가 2개까지 추가 가능
- 하단 CTA: [오늘 계획 시작하기]
- 각 항목은 “왜 추천?” 1줄이 붙는다

2) P-Action-01_ActivityDetail (추천 활동 상세)
- 예: “가벼운 산책 20분”
- 구성:
  - 상단: 소요시간/강도/주의(선택)
  - “왜 추천?”(수면/피로/활동량 기반 쉬운 설명)
  - 버튼: [YouTube로 보기] 또는 [시작하기]
  - 하단: “완료하면 체크돼요” 안내

3) P-Action-02_InProgress (진행 중)
- 타이머(선택), “일시정지/종료”
- 인터넷 약하면 오프라인 저장 배너
- 종료 시 자동으로 완료 흐름으로 이동

4) P-Action-03_Complete (완료)
- 축하/안정 톤(과한 게임화 금지)
- “어땠나요?” 1문항(이모지 5점)
- “오늘 몸 느낌” (좋음/보통/피곤) 3택1 (선택)
- [저장] 누르면 홈의 리듬 카드에 반영(“오늘 계획 1개 완료”)

5) P-Reco-YouTube-01_QueryBuilder (검색어 생성 근거)
- 상단: “오늘 추천 영상”
- 검색어 칩 3개(자동):
  예: “10분 호흡 명상”, “시니어 스트레칭 5분”, “잠 잘 오는 저녁 루틴”
- 각 칩 아래 “왜 이 검색어?” 1줄(필수)
- 버튼: [이 검색어로 보기] → 결과 화면

6) P-Reco-YouTube-02_ResultList
- CMP/Card/YouTubeResult 리스트 8~12개
- 상단 고정 안내: “외부(YouTube)로 이동합니다.”
- 필터 칩: “짧게(5~10분)” “가벼움” “의자운동” 등(의료 주장 금지)
- 빈 결과 상태(noResult)도 포함

7) P-Food-01_TodayIdeas (식단 추천 — 의료표현 금지)
- “오늘의 식사 아이디어(아침/점심/저녁)”
- 각 카드: 메뉴 1~2개 + 포인트(따뜻함/가벼움/제철) 1줄
- 버튼: [레시피 영상(YouTube)] [장보기 메모]
- “치료/예방” 표현 금지: “캠프 연계 루틴 식단” 톤만 사용
- (선택) “캠프 식단을 집에서도 이어가요” 배너(구매/결제 없음, ‘관심 등록’만)

8) P-Report-Daily-Weekly-02_Insight
- 주간 리듬 리포트 디테일:
  - 수면 규칙성/활동/스트레스 추세(미니 차트)
  - “이번 주 가장 도움이 된 것” 1개
  - “다음 주 한 가지 목표” 1개(선택 저장)
  - 캠프 유도 배너(부드럽게): [리듬 리셋 캠프 둘러보기]

[필수 상태 화면]
- 워치 미연결/권한 미허용: “워치 없이도 일부 기록 가능” 문구 + [연결하기]
- 동기화 지연: 마지막 동기화 시간 + [지금 동기화]
- 추천 불러오기 실패: [다시 시도] + 오프라인 안내
